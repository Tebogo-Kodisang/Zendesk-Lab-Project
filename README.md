# Zendesk Lab Project

This project shows hands-on experience with the Zendesk ticketing system.  
It demonstrates three simple steps of customer support:

1. **How customers reach IT** – showing how clients can submit a ticket.  
2. **How I handle requests** – reviewing incoming requests, organizing them, and responding professionally.  
3. **How I solve issues** – resolving the request and leaving notes so the team knows what was done or what to continue with.
